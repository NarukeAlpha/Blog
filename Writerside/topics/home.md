# Signal & Static

A Writerside blog that publishes from a desktop studio straight into GitHub Pages.

## What lives here

- Journal posts become Writerside topics the moment they are published from the Electron app.
- Git pushes trigger a GitHub Actions workflow that rebuilds the public site.
- Bookmarks are researched through OpenCode before they land in the reading queue.
